Alexa Facts Skill Blueprint

The Alexa Fact Skill Blueprint is a headless content project that backs an Alexa Skill. It allows you to manage content regarding facts and events that occurred on a specific day and year. This project demonstrates how to manage headless content for consumption via digital assistant and provides out-of-the-box API-based responses structured and formatted for Alexa.

Features:

  • Manage content regarding facts and events that occurred on a specific day and year
  • Demonstrates how to manage headless content for consumption via digital assistant
  • Provides out-of-the-box API based responses structured and formatted for Alexa

Technologies Used:

  • Amazon Alexa
  • CrafterCMS
  • Headless CMS
  • API Integration
